# 第二章 大模型基础与上下文窗口

要精通上下文工程，首先必须深入理解大语言模型的工作机制，特别是上下文窗口的本质。上下文窗口既是大模型的核心能力所在，也是上下文工程需要优化和管理的关键资源。

本章将从技术原理层面解析大模型如何处理上下文，Token 的概念与计算方式，以及不同模型在上下文处理能力上的差异。这些知识是理解后续技术策略的必要基础。
