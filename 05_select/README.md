# 第五章 上下文选择与检索

当信息存储在外部系统后，下一个关键问题是：如何从海量信息中精准获取当前任务所需的内容？**选择策略**正是解决这一问题的核心方法，而[检索增强生成](5.1_rag_principles.md)（[RAG](5.1_rag_principles.md)）是其最重要的实现形式。

本章将深入探讨 RAG 的基本原理、文档分块策略、嵌入模型与语义搜索、重排序技术，以及混合检索等高级 RAG 技术。
