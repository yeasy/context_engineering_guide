# 第九章 智能体上下文管理

**AI 智能体**（Agent）是能够自主规划和执行多步骤任务的系统。与简单的问答不同，智能体需要在较长时间跨度内持续工作，这对上下文管理提出了更高要求。

本章将探讨智能体架构中的上下文角色、单智能体和多智能体的上下文管理策略，以及智能体的记忆与学习机制。
