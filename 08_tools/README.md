# 第八章 工具调用与能力扩展

大语言模型的能力有边界——它无法直接访问实时数据、执行代码或操作外部系统。**工具调用**（Tool Calling / Function Calling）机制打破了这一限制，让模型能够调用外部工具来扩展能力。

本章会探讨工具调用的机制、工具设计原则、执行与结果处理，以及 MCP 协议等工具标准化方案。工具调用是构建强大 AI 应用和智能体的基础能力。
