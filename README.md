# 大模型上下文工程权威指南

> 从理论到实践，掌握 AI 时代最核心的工程能力

---

## 本书简介

**上下文工程**（Context Engineering）是大模型应用开发中最关键的核心能力之一。它不仅仅是"提示词工程"的升级版，而是一门系统性的工程学科，涵盖了如何设计、组织、优化和管理大模型所需的全部信息环境。

随着大语言模型（LLM）从实验性技术走向生产级应用，简单的提示词设计已无法满足复杂任务的需求。上下文工程应运而生，它关注的是：如何在正确的时间，将正确的信息，以正确的格式，提供给模型——从而实现可靠、高效、可扩展的 AI 系统。

本书系统性地介绍上下文工程的核心概念、技术原理、实践方法与前沿进展，帮助读者从入门到精通，成为这一领域的专家。

## 你将学到什么

通过阅读本书，你将掌握：

### 理论基础
- 上下文工程的定义、起源与发展历程
- 上下文工程与提示词工程的关系与区别
- 上下文窗口的工作原理与限制

### 核心技术
- **四大核心策略**：写入（Write）、选择（Select）、压缩（Compress）、隔离（Isolate）
- **检索增强生成（RAG）**：分块、嵌入、重排序的高级技术
- **记忆架构**：工作记忆、短期记忆、长期记忆、语义记忆、情景记忆
- **工具与函数调用**：扩展模型能力的关键机制

### 实践方法
- 结构化上下文设计（XML 标签、指令分层）
- 长文档处理与多文档管理
- 对话历史管理与上下文压缩
- 动态上下文构建与实时检索

### 进阶主题
- 多智能体系统中的上下文传递与编排
- Model Context Protocol（MCP）标准协议
- 上下文工程的可观测性与调试
- 生产环境最佳实践与性能优化

### 前沿探索
- Agentic RAG 与 Graph RAG
- 自适应上下文管理
- 上下文工程的未来发展趋势
